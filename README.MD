# Real Epic Card Battle

> by Duany Espindola

## Summary

This will be an online, turn-based card game where the goal is to bring your opponent’s life points down to zero.

I’m using this project as a way to learn new technologies, like Go for the backend.

For the frontend, I’d like to use Flutter, but I don’t know it yet. So, to get things up and running, I’ll start with Vue.js for a simple MVP (with a very basic design) just to make sure everything works. Once that’s in place, I’ll migrate to Flutter.
